# @lumen/ui

React + TypeScript + Tailwind components built on `@lumen/tokens`.

- `src/primitives/` — Button, ButtonGroup, TextLink, Badge, Avatar, Input,
  Checkbox, Radio, Switch, Select, Tabs, Tooltip
- `src/composite/` — Card, Modal, DataTable, Pagination, FormField,
  EmptyState, Toast
- `src/layout/` — Container, Stack, Grid, AppShell

Every component styles itself exclusively with semantic tokens (CSS
variables from `@lumen/tokens`) — see `docs/accessibility.md` and
`CONTRIBUTING.md` at the repo root before adding or changing a component.
